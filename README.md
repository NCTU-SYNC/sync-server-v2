# SYNC-Server-V2

## API Document

[Swagger UI](https://nctu-sync.github.io/sync-server-v2/)

To use Swagger UI locally, run `yarn openapi`.

## Installation

1. Clone the repo

    ```sh
    git clone https://github.com/NCTU-SYNC/sync-server-v2.git
    ```

2. Install YARN packages

    ```sh
    yarn install
    ```

3. Run dev server

    ```sh
    yarn serve
    ```
